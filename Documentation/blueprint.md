# **App Name**: Zwivhuyaex CV

## Core Features:

- CV Revamp: Revamp existing CV by intelligently rewriting the content, enhancing tone, structure, and format. The tool may optionally expand sections to enhance completeness.
- Job-Specific Optimization: Optimize CV for specific job descriptions by extracting keywords, matching content, and rewriting accordingly. The tool will only add information in the event that there is a significant and obvious deficiency in skills shown in the resume, as compared to the job description.
- CV Upload/Paste: Enable users to upload their CV in PDF/Word formats or paste text directly into the platform.
- Real-Time Editor: Provide a WYSIWYG editor (like Google Docs) for users to make real-time edits to their resumes.
- Template Selection: Offer multiple resume templates catering to various industries and styles.
- CV Download: Implement a download feature allowing users to download their CV in PDF format with a watermark (free) or without a watermark after payment.
- User authentication: Provide sign up / sign in using email, and also Google

## Style Guidelines:

- Primary color: Soft teal (#73D8C8) for a fresh, modern, and trustworthy feel, nodding subtly to South Africa's varied landscapes without being too overt.
- Background color: Light grey (#F2F4F3), providing a neutral and clean backdrop that ensures readability and focus on the content.
- Accent color: Coral (#F08080) as a contrasting accent for interactive elements and calls to action, drawing the user's eye and injecting warmth into the design.
- Headline font: 'Belleza' sans-serif for headlines; body font: 'Alegreya' serif for body text, to convey professionalism and design sensibility.
- Use clean and professional icons to represent different sections and functionalities. Favor minimalist, outlined style icons in the accent color.
- Design a clean and structured layout with clear visual hierarchy, ensuring optimal readability and user experience. Prioritize a mobile-first approach for accessibility.
- Incorporate subtle animations for transitions and feedback, enhancing the user experience without being distracting.
